# Reign

**© PURPLE SWING, 2026**

---

### About

Reign — created by [Purple Swing](https://github.com/Purple-Swing) — is a lightweight and performant group of systems and assets that aid Unity game development by adding built-in features to speed up development.

---

### Features

- Discord RPC support
- Expandable save file system
- Easily managable audio system
- Flexible and editable input system
- Scene loading system (with transitions)
- Large built-in utilities and libraries
