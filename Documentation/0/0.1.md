# 0.1 - Introduction

### What is Reign?

Reign, created by **Purple Swing**, is a Unity game development tool that helps by providing systems and utilities to aid game development. It was originally created in December of 2025 by Tureen, a member of Purple Swing, to help with developing games after they realised it was wasteful to recreate the same backend for every single game.

### Why Use Reign?

Reign handles object pooling, Discord RPC, an audio system, an easy-to-use input system which utilises the new Input System package, asynchronous scene loading and game saving/loading, utility classes, asset libraries, an event bus, query bus and much more to come. It helps to set up a highly advanced project by providing you with essential and configurable systems.
