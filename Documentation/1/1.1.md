# 1.1 - Installing Reign

### Installation Instructions

- Download the latest release from the [GitHub releases page](https://github.com/Purple-Swing/Reign/releases), and extract the archive.
- Create a new Unity project in Unity Editor version [*6000.4.4f1*](https://unity.com/releases/editor/whats-new/6000.4.4f1) (or [*2022.3.62f3*](https://unity.com/releases/editor/whats-new/2022.3.62f3) for legacy versions)
- Import the contents of the release folder into the Unity project folder, where the project solution is located.

Reign uses the Universal Render Pipeline, it provides its own asset by default.

---

### Porting to Newer Versions
When porting up to a new version of Reign, it is highly recommended that you delete all the project's files and then choose which files to replace or keep when merging as to not break your project functionality. For example, overwriting your GameData class may result in issues with your save games or compile errors in scripts that require fields from it. Another example may be when assets move directories which creates duplicates if the files weren't deleted.
