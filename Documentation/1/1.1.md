# 1.1 - Installing Reign

**These are the instructions for installing Reign in a new Unity project:**

**For 0.5.0.1 and prior**
- Download the latest release from the [GitHub releases page](https://github.com/Purple-Swing/Reign/releases), and extract the zip.
- Create a new Unity project in Unity Editor version [*2022.3.62f3*](https://unity.com/releases/editor/whats-new/2022.3.62f3)
- Import the contents of the release folder into the Unity project folder (where the .sln is located.)

**For 1.0.0.0 and up**
- Download the latest release from the [GitHub releases page](https://github.com/Purple-Swing/Reign/releases), and extract the zip.
- Create a new Unity project in Unity Editor version [*6000.4.4f1*](https://unity.com/releases/editor/whats-new/6000.4.4f1).
- Import the contents of the release folder into the Assets folder.

---

### Pulling Upstream
It is highly recommended that you should create an upstream of Reign in your own project so that you can pull new commits, it is also highly recommended that you choose which files to replace when merging as to not break your project functionality.
