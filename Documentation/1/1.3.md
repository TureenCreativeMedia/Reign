# 1.3 - Design

Reign's design follows a simple philosophy that it should always be persistent throughout the game's runtime using its prefab.

Reign is made up of [systems](1.2.md), each requiring their own requisites.

---

### Reign Prefab
The Reign prefab is a **DDOL (Don't Destroy On Load) Singleton** which is present in every scene until the game is quit, as soon as it is active and in the current scene.
It contains a child object with all of the systems as scripts.
